Module: IDG Publish Form

Description
===========
IDG Publish form display in content listing in IDG home page. Content push to microsite from idgnews.

Requirements
============
when click "Publish to Microsite" Content push to microsite from display.

Installation
============
Module install in idgnews.

Usage
=====
1 - Go to IDG home page.
2 - Click on "Publish to Microsite" in any node.
3 - "Publish On Microsite" form is display. four image are display in form. Click on any image and display content category select list.
4 - Select any category in any micro site image and click on publish. content push on respected microsite.
