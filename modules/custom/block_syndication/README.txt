
Module: Block Syndication.

Description
===========
Create Editor pick content in any micro-site and block create in other micro-site.

Requirements
============

Editor pick content need to check "Publish To Other Sites" checkbox.

Installation
============
Module install in all micro-site.

Usage
=====
1 - Create Editor pick content type in any micro-site and check "Publish To Other Sites" block is create to other micro site.
2 - Edit Editor pick content type in any micro-site, block also be updated
in other micro-site.
