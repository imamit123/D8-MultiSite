Module: IDG xml

Description
===========
This Modle describes the mapping of IDG NS XML fields to the fields in the IDGNews.in CMS. The feed consists of two XML’s – the main XML and the story XML. The main XML contains a listing of all the stories. The story XML contains the details of each story.

The IDG NS main XML feed is available at https://www.idgns.com/feeds/partner/idgns. A sample story XML feed can be taken from https://www.idgns.com/feeds/partner/idgns/article.do?id=3052325 for the purpose of this document.

Installation
============
Module install in idgnews.

Usage
=====
Get data from xml and create node.
